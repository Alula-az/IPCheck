    Column: Device Product / DeviceProduct Vendor Filter
    Keep: Logs containing allowed vendors (...).

    Column: Attacker IP / Attacker Address Network Security

    Keep: Logs where the Manager Receipt Time date matches or falls within the End Time date range (e.g. if the End Time spans Jul 14 to Jul 15, the Manager Receipt date must be within this range).

    Sorting Similarity: Sorts descending (newest first) by End Time, falling back to Manager Receipt Time.
    Keep Range: Keeps rows between the newest day's first closed;hidden event (upper limit) and yesterday's first closed;hidden event (lower limit).
    Discard Limits: Discards events newer than today's limit, and events equal to or older than yesterday's limit, evaluating both time columns to prevent leaks.
